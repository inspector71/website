LibreELEC USB-SD Creator

To create a bootable LibreELEC instance, on a USB or SD Card, for any LibreELEC-supported platform, please download and run (as Administrator) the LibreELEC USB-SD Creator app.

Need help using this app? A step by step guide is available on our documentation site.

  • Windows
  • MacOS (currently not available)
  • Linux 32Bit (currently not available)
  • Linux 64Bit (currently not available)
